zpArdoise 1.0 – a ZenPhoto/ZenPage theme

This is the first release of this theme. Please report bugs on the forum and I will try to fix as soon as possible.

zpArdoise theme is a theme for ZenPhoto/ZenPage, inherited from zpGalleriffic theme (thanks to gjr).

The theme has been optimized for 150x150 px squared thumbnails (five thumbnails per line): think of adjusting the size of thumbnails, otherwise you will have results that are not very elegant.
You can also use the script Galleriffic (presentation script of images from the gallery). In this case, you should not have any albums that mixes sub-albums and pictures, because of configuration of the theme, album.php can display two pagination bars (that of the theme and that of gallerific script).

The additional scripts used are the same that zpGalleriffic:

  • Galleriffic, jQuery History Plugin, Slide and Toggle Fade

I also added a script for displaying the images with the arrow keys.

The theme supports the following ZenPhoto plugins:

  • comment_form, contact_form, image_album_statistics, print_album_menu, rating, slideshow, user_login-out, zenpage

Installation

Important: You must have ZenPhoto 1.2.9 release or higher.

  1. Upload the zip file to your computer
  2. Unzip the downloaded zip file locally, and upload the zpArdoise folder to the directory « /themes/ » of your ZenPhoto site
  3. In ZenPhoto administration, go to the Themes tab and activate the zpArdoise theme
  4. Navigate to Options>Theme to view and configure the available options for zpArdoise

Optional:

  1. You can customize the header image by placing images 900 px wide (eg 900x120 px) in the directory « themes/zpArdoise/images/ », and configure your header in Options>Theme
  2. You can use the RSS icon provided by copying the file « themes/zpArdoise/images/rss.gif » in the directory « zp-core/images/ »

Thanks in advance for your feedback and enjoy with this theme!

zpArdoise 1.0 – un thème ZenPhoto/ZenPage

Il s'agit de la première version de ce thème. Rapportez les bugs sur le forum ZenPhoto.org et j’essaierais de les corriger aussitôt que possible.

zpArdoise est un thème pour ZenPhoto/ZenPage directement hérité du thème zpGalleriffic (merci à gjr).

Le thème a été optimisé pour des vignettes carrées de 150x150 px (5 vignettes par ligne) : pensez à régler la taille des vignettes car sinon vous aurez des résultats qui ne seront pas très élégants.
Vous pouvez également utiliser le script Galleriffic (script de présentation des images de la galerie). Dans ce cas, vous ne devriez pas avoir d'albums qui mélangent des sous-albums et des photos, car en fonction de la configuration de votre thème, album.php peut afficher deux barres de pagination (celle du thème et celle du script gallerific).

Les scripts additionnels utilisés sont les mêmes que zpGalleriffic :

  • Galleriffic, jQuery History Plugin, Slide and Toggle Fade

J'ai également ajouté un script de visualisation des images avec les flèches du clavier.

Le thème supporte les plugins ZenPhoto suivants :

  • comment_form, contact_form, image_album_statistics, print_album_menu, rating, slideshow, user_login-out, zenpage

Installation

Important : Vous devez avoir la version ZenPhoto 1.2.9 ou supérieure.

  1. Téléchargez le fichier zip sur votre ordinateur
  2. Décompressez le fichier zip localement, et télécharger le dossier zpArdoise dans votre répertoire « /themes/ » de votre site ZenPhoto
  3. Dans l'administration ZenPhoto, allez sur l'onglet Thèmes et activer le thème zpArdoise
  4. Allez sur Options>Thème afin d'afficher et configurer les options disponibles pour zpArdoise

Optionnel :

  1. Vous pouvez personnaliser l'image d’entête en déposant des images d’une largeur de 900 px (900 x 120 px par exemple) dans le répertoire « themes/zpArdoise/images/ », et en configurant le header dans Options>Thème
  2. Vous pouvez utiliser l’icône RSS fournie dans le thème en copiant le fichier « themes/zpArdoise/images/rss.gif » dans le répertoire « zp-core/images/ »

Merci d'avance pour vos avis et commentaires, et profitez bien de ce thème !
