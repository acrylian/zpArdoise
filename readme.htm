zpArdoise 1.4 – a ZenPhoto/ZenPage theme

Please report bugs on the forum and I will try to fix as soon as possible.

zpArdoise theme is a theme for ZenPhoto/ZenPage, inherited from zpGalleriffic theme (thanks to gjr).
Feel free to download and use it, and thanks in advance for your feedback!

Description

Important: To use the release 1.4 of the theme, you must have ZenPhoto 1.4.1 (or more).
If you have ZenPhoto from 1.4 to 1.4.0.4, you should use zpArdoise 1.3.
If you have ZenPhoto from 1.2.9 to 1.3.1.2, you should use zpArdoise 1.0.

Additional scripts used:

  • Galleriffic, jQuery History Plugin, Slide and Toggle Fade
  • script for displaying the images with the arrow keys.

The theme supports the following ZenPhoto plugins:

  • colorbox, comment_form, contact_form, flag_thumbnail, GoogleMap, image_album_statistics, print_album_menu, rating, register_user, slideshow, user_login-out, zenpage

Improvements of 1.4 release

  • With option «Use Galleriffic script», displays the description of the image and its EXIF ​​data (if «Show image EXIF ​​data» is choosen)
  • Better integration between the scripts galleriffic and colorbox
  • Better integration between the options «Show Image Statistic strip» and «Use Colorbox Album»
  • Using the login form of ZenPhoto
  • Some graphical improvements
  • Fix some issues (read changelog.txt)

Installation

  1. Upload the zip file to your computer
  2. Unzip the downloaded zip file locally, and upload the zpArdoise folder to the directory « /themes/ » of your ZenPhoto site
  3. In ZenPhoto administration, go to the Themes tab and activate the zpArdoise theme
  4. Navigate to Options>Theme to view and configure the available options for zpArdoise
  5. In Options>Theme, adujst squared thumb size to 150 px (the theme has been optimized for this size)

Options

  1. You can customize the header image by placing images 900 px wide (eg 900 px x 120 px) in the directory « themes/zpArdoise/images/ », and configure your header in Options>Theme
  2. You can use the script Galleriffic (presentation script of images). Then, you should not have any albums that mixes sub-albums and pictures (in this case, all sub-albums are displayed on album.php, to avoid having two navigation bars, that of the zpardoise theme and that of the galleriffic script)
  3. You can use the RSS icon provided by copying the file « themes/zpArdoise/images/rss.gif » in the directory « zp-core/images/ »

 

 

English version - Version Française

zpArdoise 1.4 – un thème ZenPhoto/ZenPage

Rapportez les bugs sur le forum ZenPhoto.org et j’essaierais de les corriger aussitôt que possible.

zpArdoise est un thème pour ZenPhoto/ZenPage hérité du thème zpGalleriffic (merci à gjr).
N'hésitez pas à le télécharger et à l'utiliser, et merci d'avance pour vos avis et commentaires !

Description

Important : Pour utiliser la version 1.4 du theme, vous devez utiliser ZenPhoto 1.4.1 (ou plus).
Si vous utilisez ZenPhoto allant de 1.4 à 1.4.0.4, vous devez utiliser zpArdoise 1.3.
Si vous utilisez ZenPhoto allant de 1.2.9 à 1.3.1.2, vous devez utiliser zpArdoise 1.0.

Scripts additionnels utilisés :

  • Galleriffic, jQuery History Plugin, Slide and Toggle Fade
  • script de visualisation des images avec les flèches du clavier.

Le thème supporte les plugins ZenPhoto suivants :

  • colorbox, comment_form, contact_form, flag_thumbnail, GoogleMap, image_album_statistics, print_album_menu, rating, register_user, slideshow, user_login-out, zenpage

Améliorations de la version 1.4

  • Avec l'option «Use Galleriffic script», affiche la description de l'image et ses données EXIF (si l'option «Show image EXIF data» est choisie)
  • Meilleure intégration entre les scripts galleriffic et colorbox
  • Meilleure intégration entre les options «Show Image Statistic strip» et «Use Colorbox Album»
  • Utilisation du formulaire de connexion proposé par ZenPhoto
  • Quelques améliorations graphiques
  • Correction de quelques bugs (voir changelog.txt)

Installation

  1. Téléchargez le fichier zip sur votre ordinateur
  2. Décompressez le fichier zip localement, et télécharger le dossier zpArdoise dans votre répertoire « /themes/ » de votre site ZenPhoto
  3. Dans l'administration ZenPhoto, allez sur l'onglet Thèmes et activer le thème zpArdoise
  4. Allez sur Options>Thème afin d'afficher et configurer les options disponibles pour zpArdoise
  5. sur Options>Thème, ajustez la taille des vignettes carrées à 150 px (le thème a été optimisé pour cette taille)

Options

  1. Vous pouvez personnaliser l'image d’entête en déposant des images d’une largeur de 900 px (900 px x 120 px par exemple) dans le répertoire « themes/zpArdoise/images/ », et en configurant le header dans Options>Thème
  2. Vous pouvez utiliser le script Galleriffic (script de présentation des images). Vous devriez alors éviter d'avoir des albums qui mélangent des sous-albums et des photos (dans ce cas, tous les sous-albums sont affichés, pour ne pas avoir deux barres de navigation, celle du thème zpArdoise et celle du script gallerific)
  3. Vous pouvez utiliser l’icône RSS fournie dans le thème en copiant le fichier « themes/zpArdoise/images/rss.png » dans le répertoire « zp-core/images/ »
